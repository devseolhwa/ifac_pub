  • API 유형
    주요 콘텐츠인 인천시·군·구에서 열리는 문화예술 행사 데이터 API입니다.
  • 트래픽
    5,000건 / 월
  • 사용환경*
  • 서비스 이용약관
    • 제 1 조 [목적]이 이용약관(이하'약관'이라 합니다)은인천문화예술 통합DB와 국민 간에 인천문화예술 통합DB이 제공하는 인천문화예술 통합DB Open API 서비스의 이용에 관한 제반 사항과 기타 필요한 사항을 규정함을 목적으로 합니다.
    • 제 2 조 [용어의 정의]
      • 이 약관에서 사용하는 용어의 정의는 다음 각호와 같습니다.
        Open API 서비스라 함은 국민이 자발적인 참여를 통해 자유롭게 정보를 공유하고 창조적인 서비스를 생산할 수 있도록 하기 위하여 인천문화예술 통합DB이 제공•운영하는 문화행사,문화시설,문화재,지역축제 Open API서비스를 말합니다.
        API라 함은 Application Programming Interface의 약자로서 인천문화예술 통합DB Open API 서비스를 국민이 자신이 구축한 사이트에서 자유롭게 사용할 수 있도록 인천문화예술 통합DB이 제공하는 등록 값의 집합을 말합니다.
        Open API 서비스라 함은 인천문화예술 통합DB이 운영하는 API(Application Programming Interface)를 국민에게 개방하거나 제공하는 등의 서비스를 말합니다.
        인증Key라 함은 인천문화예술 통합DB의 관련 시스템이 API 서비스 이용허가를 받은 사람임을 식별할 수 있도록 인천문화예술 통합DB이 국민에게 개별적으로 할당하는 고유한 값을 말합니다.
      • 이 약관에서 정하지 아니한 것은 관계 법령 및 서비스안내에서 정하는 바에 따르며, 그 외에는 일반 관례에 따릅니다.
    • 제 3 조 [적용범위]
      • 본 약관은 인천문화예술 통합DB 홈페이지(iq.ifac.or.kr) 회원에 가입한 후, 본 약관에 동의한 자(이하 “회원”이라 합니다) 에 대하여 적용합니다.
      • 본 약관은 회원에 대한 Open API 서비스 제공행위 및 회원의 Open API 서비스 이용행위에 대하여 우선적으로 적용됩니다.
      • 본 약관에서 규정하지 않은 인증Key 발급 등 회원 관리에 관한 제반사항은 인천문화예술 통합DB 통합회원 관리정책을 준용합니다.
    • 제 4 조 [이용약관의 효력 및 변경]
      • 본 이용약관은 서비스의 이용을 위하여 회원이 동의를 함으로써 효력이 발생합니다.
      • 인천문화예술 통합DB는 합리적인 사유가 발생할 경우 본 약관을 변경할 수 있으며, 이 경우 일정한 기간을 정하여 적용일자 및 변경사유를 명시한 사항을 회원에게 공지 또는 통지합니다.
      • 제2항에 따른 약관의 변경은 회원이 동의함으로써 그 효력이 발생됩니다. 다만, 제2항에 따른 통지를 하면서 회원에게 일정한 기간 내에 의사표시를 하지 않으면 의사표시가 표명된 것으로 본다는 뜻을 명확히 전달하였음에도 회원이 명시적으로 거부의 의사표시를 하지 아니한 경우에는 회원이개정약관에 동의한 것으로 봅니다.
      • 약관의 변경 사실 및 내역을 확인하지 못하여 발생한 모든 손해에 대한 책임은 회원에게 귀속됩니다.
    • 제 5 조 [Open API 서비스의 이용]
      • Open API의 모든 서비스는 본 약관에 동의한 회원에 한하여 제공합니다.
    • 제 6 조 [Open API 서비스의 제한]
      • 인천문화예술 통합DB은 특정 Open API 서비스의 범위를 제한하거나 별도의 이용 가능 시간 또는 이용 가능 횟수를 지정할 수 있으며, 관련 법률 개정으로 인해 Open API 서비스 제공 대상이 변경되어 더 이상 활용을 할 수 없을 경우, 서비스 활용으로 인해 제공기관의 업무에 지장을 초래하거나 인프라 성능 등의 이유로 서비스 제공 상의 성능 문제가 발생한 경우 활용을 제한할 수 있습니다. 이 경우 이를 회원에게 사전에 고지하여야 합니다.
      • 인천문화예술 통합DB는 회원이 Open API 서비스를 이용함에 있어 법령을 위반하거나 약관 또는 서비스 이용기준 등을 위반한 경우, 제공된 정보를 임의로 위조•변조하여 저작권을 위반하는 경우에는 제1항의 규정에도 불구하고 즉시 인증Key의 이용을 정지하는 등의 조치를 취할 수 있습니다.
      • 인천문화예술 통합DB은 회원이 Open API 서비스에 대한 불법적인 해킹 시도, 비정상적인 방식을 통한 오남용 시도, 네트워크 사용 초과 등의 시도를 하는 경우 제1항의 규정에도 불구하고 즉시 인증Key의 사용을 정지시킬 수 있습니다.
    • 제 7 조 [인증Key의 이용 및 관리]
      • 회원은 발급 받은 인증Key를 타인에게 제공, 공개하거나 공유할 수 없으며, 발급 받은 회원 본인에 한하여 이를 사용할 수 있습니다.
      • 회원은 여러개의 인증Key를 발급받아 사용할 수 있으며, 어떠한 경우에도 상업적인 용도로 사용하여서는 안됩니다.
      • 인천문화예술 통합DB은 인증Key를 발급함에 있어 이용기간을 지정할 수 있으며, 이용기간을 변경하고자 하는 경우에는 사전에 이를 고지하여야 합니다.
    • 제 8 조 [Open API 서비스 이용시의 주의사항]
      • 인천문화예술 통합DB는 관계법령의 제•개정 및 기타 정책적 사유 등에 따라 Open API 서비스를 변경하거나 중단할 수 있습니다.
      • 인천문화예술 통합DB는 Open API 서비스를 운영함에 있어 데이터의 특정범위를 분할하거나 또는 전체에 대하여 별도의 이용가능 시간 또는 이용가능 횟수를 지정할 수 있으며 이를 사전에 고지하여야 합니다.
      • 회원은 Open API 서비스를 이용한 검색결과를 노출함에 있어 선정적, 폭력적, 혐오적인 내용을 포함하여 반사회적, 비도덕적, 불법적인 내용과 결합 또는 연계하거나 인접하도록 구성할 수 없으며, 검색결과의 공공성을 준수하여야 합니다.
      • 인천문화예술 통합DB는 Open API 서비스를 이용한 검색결과와 함께 광고를 게재할 권리를 가집니다. 다만 광고를 게재하고자 할 경우 사전에 회원 또는 비회원에게 이를 공지 또는 통지합니다.
      • 인천문화예술 통합DB는 개인정보 보호정책을 공시하고 준수합니다.
    • 제 9 조 [회원 또는 비회원의 의무]
      • 회원은 Open API 서비스를 이용함에 있어서 본 약관에서 규정하는 사항과 기타 인천문화예술 통합DB이 정한 제반 규정, 공지사항 및 관계 법령을 준수하여야 하며, 인천문화예술 통합DB의 업무에 방해가 되는 행위 또는 인천문화예술 통합DB의 명예를 손상시키는 행위를 해서는 안됩니다.
      • 인천문화예술 통합DB의 Open API 서비스를 이용함에 있어서 회원 행위에 대한 모든 책임은 당사자가 부담하며, 회원은 인천문화예술 통합DB을 대리하는 것으로 오해가 될 수 있는 행위를 해서는 안됩니다.
    • 제 10 조 [Open API 서비스 저작권]
      • 회원은 Open API 서비스 이용 시 인천문화예술 통합DB 및 제3자의 지적재산권을 침해해 서는 안됩니다.
      • 인천문화예술 통합DB이 제공하는 Open API 데이터 검색결과 등에 대한 저작권은 인천문화예술 통합DB에 있고, 인천문화예술 통합DB의 이용허락으로 인해 회원이 당해 API 및 데이터파일, 검색결과 등에 대한 저작권을 취득하는 것은 아닙니다. 다만, 회원이 제작한 프로그램에 대한 저작권은 회원에게 귀속됩니다.
      • 회원은 Open API 서비스를 이용하여 검색결과를 노출할 경우, 해당 페이지에 “인천문화예술 통합DB Open API"를 사용한 결과임을 명시해야 합니다. 다만, 인천문화예술 통합DB이 별도의 표시방식을 정한 경우에는 그에 따라야 합니다.
    • 제 11 조 [API 및 데이터파일 이용허락 조건]
      • 인천문화예술 통합DB는 Open API 서비스에 대하여 저작자 및 출처 표시 조건으로 자유이용을 허락함을 원칙으로 합니다.
    • 제 12 조 [책임의 제한]
      • 인천문화예술 통합DB는 Open API 서비스에 관하여 약관, 서비스별 안내, 기타 인천문화예술 통합DB이 정한 이용기준 및 관계법령을 준수하지 않은 이용으로 인한 결과에 대하여 책임을 지지 않습니다.
      • 인천문화예술 통합DB는 Open API 서비스의 사용불능으로 인하여 회원 또는 비회원에게 발생한 손해에 대하여 책임을 지지 않습니다.
      • 인천문화예술 통합DB는 회원이 인천문화예술 통합DB Open API 서비스를 이용하여 기대하는 수익을 얻지 못하거나 상실한 것에 대하여 책임을 지지 않습니다.
      • 인천문화예술 통합DB는 회원과 제3자 상호 간에 Open API 서비스를 매개로 발생한 분쟁에 대해 개입할 의무가 없으며, 이로 인한 손해를 배상할 책임도 없습니다.
    • 제 13 조 [이용자격 박탈 및 손해배상]
      • 인천문화예술 통합DB는 회원이 본 이용약관을 준수하지 않는 경우 서비스 사용 중지 및 이용자격을 박탈할 수 있습니다.
      • Open API 서비스 이용상 회원의 귀책사유로 인하여 인천문화예술 통합DB에 손해가 발생한 경우 인천문화예술 통합DB는 본 약관에 따른 계약의 해지와는 별도로 손해배상을 청구할 수 있습니다.
      • 인천문화예술 통합DB Open API 서비스의 이용으로 인천문화예술 통합DB과 회원간에 발생한 분쟁에 관하여 소송이 제기되는 경우 각 당사자는 자신의 주소지를 관할하는 법원에 소송을 제기할 수 있습니다.
    • <부칙> 제 1 조 (시행일) 본 약관은 2014년 11월 14일부터 적용됩니다.
